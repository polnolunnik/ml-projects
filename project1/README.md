# ml-projects
# Прогнозирование времени поездки такси в Нью-Йорке

## Описание задачи

Построить модель машинного обучения, которая предсказывает длительность поездки такси (в секундах) на основе данных.

Данные взяты из соревнования Kaggle NYC Taxi Trip Duration: https://www.kaggle.com/competitions/nyc-taxi-trip-duration/data

## Данные

Использована выборка из 50 000 строк из тренировочного набора.

Основные признаки:
- pickup_datetime — дата и время посадки
- pickup_longitude / pickup_latitude — координаты посадки
- dropoff_longitude / dropoff_latitude — координаты высадки
- vendor_id — компания
- trip_duration — целевая переменная (в секундах)

## Что сделано в проекте

1. EDA (исследовательский анализ данных)
   - Визуализация распределения длительности поездок
   - Выявление выбросов через гистограмму и boxplot
   - Удаление выбросов (поездки короче 60 секунд и длиннее 2500 секунд)

2. Feature Engineering (создание новых признаков)
   - Из времени посадки: час, день недели, месяц
   - Бинарные признаки: выходной, ночь
   - Расстояние между точками посадки и высадки (формула гаверсинуса)

3. Подготовка данных
   - Масштабирование числовых признаков (StandardScaler)
   - Кодирование категориального признака vendor_id (OneHotEncoder)
   - Разделение на train/test (80% / 20%)

4. Обучение и сравнение 9 моделей
   - Linear Regression, Ridge, Lasso
   - k-NN
   - Random Forest
   - Gradient Boosting (sklearn)
   - XGBoost, LightGBM, CatBoost

5. Дополнительный анализ
   - GridSearch для подбора гиперпараметров k-NN
   - Анализ L1-регуляризации (разреживание весов)
   - Важность признаков (Random Forest)
   - PyTorch нейросеть (базовый уровень)

## Результаты

Лучшая модель: LightGBM

- RMSE: 282.5 секунд (примерно 4.7 минуты)
- R2: 0.674 (модель объясняет 67% дисперсии)

Сравнение всех моделей:

 - Linear Regression: RMSE = 344.4, R2 = 0.515
 - Ridge (L2): RMSE = 344.4, R2 = 0.515
 - Lasso (L1): RMSE = 344.4, R2 = 0.515
 - k-NN (k=5): RMSE = 309.1, R2 = 0.610
 - Random Forest: RMSE = 298.9, R2 = 0.635
 - Gradient Boosting (sklearn): RMSE = 286.4, R2 = 0.665
 - XGBoost: RMSE = 289.3, R2 = 0.658
 - LightGBM: RMSE = 282.5, R2 = 0.674
 - CatBoost: RMSE = 284.9, R2 = 0.668
