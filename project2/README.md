# Проект №2: Классификация клиентов банка (Bank Marketing)

## Описание задачи

Построить модель классификации, которая предсказывает, согласится ли клиент банка на срочный депозит на основе данных о маркетинговой кампании.

Целевая переменная: `y` — бинарный признак:
- 1 — клиент согласился на депозит
- 0 — клиент отказался

---

## Данные
- Источник:Bank Marketing (UCI / OpenML)
- Количество строк: 45 211
- Количество признаков: 17
- Дисбаланс классов: 11.7% согласий, 88.3% отказов


## Использованные методы

### 1. EDA (исследовательский анализ)
- Визуализация распределений (целевая переменная, профессии, семейное положение, образование)
- Анализ групп: кто чаще соглашается на депозит (студенты, пенсионеры, люди с высшим образованием)

### 2. Feature Engineering
- `job_length` — длина названия профессии
- `education_length` — длина названия образования
- `age_group` — возрастные группы (young, middle, senior, elderly)

### 3. Подготовка данных
- Масштабирование: StandardScaler для числовых признаков
- Кодирование:OneHotEncoder для категориальных признаков
- Разделение: 80% — обучение, 20% — тест

### 4. Модели
- Logistic Regression - Линейная
- КNN - Метрическая
- Random Forest - Ансамбль деревьев 
- LightGBM - Градиентный бустинг
- CatBoost -  Градиентный бустинг

### 5. Метрики
- ROC‑AUC  — главная метрика при дисбалансе классов
- PR‑AUC— показывает, как модель находит редкий класс
- F1 — баланс точности и полноты
- Precision, Recall, Accuracy


## Результаты

### Лучшая модель: **CatBoost**
- ROC‑AUC - 0.9334
- PR‑AUC - 0.63 
- F1 - 0.578 
- Precision - 0.65 
- Recall - 0.52 
- Accuracy - 0.91 

### Интерпретация для бизнеса
- Из 100 клиентов, которых модель назвала "целевыми", 65 реально согласятся** на депозит
- Модель находит 52 из 100 всех реальных согласившихся
- Это позволяет сократить затраты на бесполезные звонки и эффективно использовать ресурсы

### Ключевые выводы
- Самые важные признаки: длительность звонка (`duration`), профессия, образование, возраст
- Кто чаще соглашается: студенты (28.7%), пенсионеры (22.8%), люди с высшим образованием (15%)
